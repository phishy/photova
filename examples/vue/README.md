# Brighten - Vue Example

A minimal example of using Brighten with Vue 3 and TypeScript.

## Setup

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Usage

This example demonstrates:

- Using `ref` to manage the editor container
- Lifecycle hooks (`onMounted`, `onUnmounted`) for setup/cleanup
- Loading images from file input
- Exporting edited images
- TypeScript with Vue SFC
