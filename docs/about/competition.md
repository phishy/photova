# Competitive Analysis - Image Editor SDKs

Last updated: December 2024

## Market Overview

The JavaScript image editor SDK market is dominated by a few key players, most of which are paid solutions. Brighten differentiates by being **AI-native** and **open source**.

## Competitors

### Pintura (PQINA)
- **URL**: https://pqina.nl/pintura/
- **Pricing**: Paid (starts ~$99)
- **Headline**: "A powerful JavaScript Image Editor that integrates with every stack"
- **Tagline**: "A fully configurable image editor SDK that's intuitive on mobile and desktop"
- **Strengths**:
  - Mature product (since 2018)
  - Excellent documentation
  - 3,100+ companies using it
  - Strong testimonials (Wes Bos endorsement)
  - Framework adapters for React, Vue, Angular, Svelte, jQuery, Alpine
  - Video editing extension
- **Weaknesses**:
  - Paid license required
  - AI features require third-party integration (not built-in)
  - No open source option
- **Key Features**: Crop, rotate, resize, filters, annotations, watermarks, stickers, redaction

### IMG.LY (CreativeEditor SDK / PhotoEditor SDK)
- **URL**: https://img.ly/
- **Pricing**: Enterprise (contact sales)
- **Headline**: "Embed powerful photo editing into your web, iOS & Android apps"
- **Tagline**: "All with just a few lines of code"
- **Strengths**:
  - Enterprise-grade
  - Multi-platform (Web, iOS, Android, React Native, Flutter)
  - Native AI features (via plugins)
  - Extensive asset library integrations (Getty, Unsplash, Pexels)
  - Client-side processing (no server costs)
- **Weaknesses**:
  - Enterprise pricing (expensive)
  - Complex setup for smaller projects
  - AI requires separate API licenses
- **Key Features**: Full design editor, video editing, templates, animations, AI plugins

### Filerobot Image Editor (Scaleflex)
- **URL**: https://scaleflex.github.io/filerobot-image-editor/
- **Pricing**: Free / Open Source
- **Headline**: "Edit, Resize, and Filter any images for Free"
- **Strengths**:
  - Free and open source
  - Easy integration
  - Good basic features
- **Weaknesses**:
  - No AI features
  - Less polished UI
  - Fewer advanced features
  - Less active development
- **Key Features**: Crop, resize, rotate, filters, watermark, annotations

### Polotno
- **URL**: https://polotno.com/
- **Pricing**: Freemium
- **Strengths**:
  - Canvas/design editor focus
  - React-based
  - Good for template systems
- **Weaknesses**:
  - More focused on design/canvas than photo editing
  - Not specifically an image editor
- **Key Features**: Canvas editor, templates, design tools

### Syncfusion React Image Editor
- **URL**: https://syncfusion.com/react-components/react-image-editor
- **Pricing**: Part of Syncfusion suite (paid)
- **Strengths**:
  - Part of larger component library
  - Enterprise support
- **Weaknesses**:
  - Requires Syncfusion license
  - Less standalone focus
- **Key Features**: Basic editing, filters, annotations

## Brighten's Competitive Advantages

### 1. AI-Native
Unlike competitors where AI is an afterthought or plugin, Brighten has AI built into the core:
- Background removal (via Replicate)
- Object erasing/inpainting (LaMa model)
- Image enhancement/unblur
- Colorization

### 2. Open Source
- Free to use
- No license fees
- Community-driven
- Transparent codebase

### 3. Modern Architecture
- TypeScript-first
- Zero dependencies in core
- Framework agnostic
- State-of-the-art AI models

### 4. Simple Integration
- One import
- Works with React, Vue, Svelte, vanilla JS
- Drop-in UI component

## Messaging Guidelines

### Do Say:
- "AI-native" (not "AI-powered" - everyone says that)
- "Built in, not bolted on"
- "Open source alternative to Pintura"
- "State-of-the-art AI models"
- "Zero AI infrastructure required"

### Don't Say:
- "AI-powered" (overused)
- "Revolutionary" (cringe)
- "Best" without proof
