# Photova Enterprise Features

This directory contains enterprise-only features that require a valid license.

## Structure

```
ee/
├── LICENSE              # Enterprise license terms
├── README.md            # This file
└── (future features)    # SSO, Audit Logs, Advanced Analytics, etc.
```

## Usage

Enterprise features are loaded automatically when a valid license key is present:

```env
PHOTOVA_LICENSE_KEY=your-license-key
```
